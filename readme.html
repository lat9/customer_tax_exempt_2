What it does

Customer Tax Exempt v2 allows you to identify taxes from which your store's customers are exempt. Refer to the plugin's on-line documentation for additional information.

Installing and Upgrading

There are no core-file overwrites in this plugin; however, you should always backup your store's database and files prior to making any changes.

  1. Rename /YOUR_ADMIN to match the name of your Zen Cart admin directory.
  2. Upload the plugin's files to your store's file system:
    1. /includes/auto_loaders/config.CustomerTaxExempt.php
    2. /includes/classes/observers/TaxExemptCustomerObserver.php
    3. /YOUR_ADMIN/includes/auto_loaders/config.customer_tax_exempt_admin.php
    4. /YOUR_ADMIN/includes/classes/observers/CustomerTaxExemptAdminObserver.php
    5. /YOUR_ADMIN/includes/init_includes/init_customer_tax_exempt_admin.php
    6. /YOUR_ADMIN/includes/languages/english/extra_definitions/customer_tax_exempt.php
  3. Sign into your admin-console or, if you are already signed in, click the Admin Home link at the top of the page.

Un-install

Delete the plugin's added files (see above) copied during the installation. Finally, run the /uninstall_customer_tax_exempt.sql to remove the plugin's database changes.
